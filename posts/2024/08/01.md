# pnpm TypeScript

* [参考](https://www.typescriptlang.org/download/)

## `package.json`作成
::: code-group
```sh [pnpm]
pnpm init
```
:::

## `typescript`を追加
::: code-group
```sh [pnpm]
pnpm add typescript -D
```
:::

## `tsconfig.json`作成
* [tsconfig.jsonを設定する](https://typescriptbook.jp/reference/tsconfig/tsconfig.json-settings)

::: code-group
```sh [pnpm]
pnpm tsc --init
```
:::

## Hello World!
```
.
├── dist
│   └── index.js
├── node_modules
│   └── ***
├── package.json
├── pnpm-lock.yaml
├── src
│   └── index.ts
└── tsconfig.json
```

::: code-group
```ts [src/index.ts]
console.log('Hello World!')
```
:::

&emsp;`"outDir"`を指定する。
::: code-group
```json [tsconfig.json]
{
  "compilerOptions": {
    "outDir": "./dist",
    ...
  }
}
```
:::

## Command
::: code-group
```sh [pnpm]
pnpm tsc && node ./dist/index.js
```
:::
