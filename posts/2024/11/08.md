# Astro Starlight

* [Starlight](https://starlight.astro.build/)

* [Starlight Starter Kit: Basics](https://github.com/withastro/starlight/tree/main/examples/basics)

:::code-group
```sh [pnpm]
pnpm create astro@latest -- --template starlight
```
:::

```
.

│       └── docs
│           ├── guides
│           │   └── example.md

pnpm install

```
